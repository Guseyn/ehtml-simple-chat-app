# Simple [EHTML](https://github.com/Guseyn/EHTML) Chat App

```bash
git clone git@github.com:Guseyn/ehtml-simple-chat-app.git
cd ehtml-simple-chat-app
npm i
npm run app
```

Then go to http://localhost:8000/
